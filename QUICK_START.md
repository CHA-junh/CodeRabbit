# 🚀 빠른 시작 가이드

## 📋 필수 설치 (1회만)

### 1. TortoiseSVN 설치

- [TortoiseSVN 다운로드](https://tortoisesvn.net/downloads.html)
- Windows 64-bit 설치
- **재부팅 필수!**

### 2. Cursor AI 설치

- [Cursor AI 다운로드](https://cursor.sh/)
- 설치 및 실행

## 🎯 첫 번째 설정 (1회만)

### 프로젝트 가져오기

1. **C:\ 폴더에서 우클릭**
2. **TortoiseSVN → Checkout**
3. **URL**: `http://172.20.30.90/scm/svn/BIST_NEW`
4. **OK** 클릭

## 📝 매일 사용법

### 시작할 때

1. **C:\BIST_NEW 우클릭**
2. **TortoiseSVN → Update**

### 개발 중

1. **Cursor AI**에서 코드 편집
2. **C:\BIST_NEW 우클릭**
3. **TortoiseSVN → Commit**
4. **메시지 입력** 후 OK

### 끝날 때

1. **최종 커밋** 실행
2. **변경사항 확인**

## ⚠️ 주의사항

- ✅ **커밋 전 테스트** 필수
- ✅ **의미있는 메시지** 작성
- ❌ **node_modules** 커밋 금지
- ❌ **.env 파일** 커밋 금지

## 🆘 문제 발생 시

- **인증 오류**: 관리자에게 계정 문의
- **충돌**: 팀 리더에게 문의
- **연결 오류**: 네트워크 확인
